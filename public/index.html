Synthetix Contracts

This is the SNX auto redirection tool. It uses HTTP 302 temporary redirects to redirect to the latest version of the Synthetix protocol contracts on Etherscan.

Please use /ContractName (case-sensitive) to auto-redirect to the latest contract version. E.g. https://contracts.synthetix.io/Synthetix links to the latest underlying.

For the full list of contract names, please look at our developer docs.

For testnets (kovan, rinkeby and ropsten), please prefix the testnet in lowercase. E.g.

  • https://contracts.synthetix.io/kovan/Synthetix
  • https://contracts.synthetix.io/kovan/ProxyERC20
  • https://contracts.synthetix.io/kovan/SynthsUSD
  • https://contracts.synthetix.io/kovan/ProxyERC20sUSD
  • https://contracts.synthetix.io/kovan/Depot
  • https://contracts.synthetix.io/kovan/EtherCollateral
